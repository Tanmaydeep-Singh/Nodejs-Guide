# Nodejs Guide 

Hii!
In this repo you will learn to build a basic but a complete backend. 

### What you will learn?
- Nodejs
- Expressjs
- Mongoose
- EJS


### Pre-requisite
- HTML
- CSS
- Javascrypt(Basic)


### How to get packages which are used?
- clone this repo.
- open terminal.
- cd to file location.
- use "npm i" to install packages.
- use "nodemon" to run the server.
